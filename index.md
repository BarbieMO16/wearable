# Wearable interactivo

Este sitio documenta el desarrollo de mi proyecto wearable.

## Objetivo
Diseñar una prenda interactiva que responda a estímulos mediante sensores y actuadores.

## Componentes utilizados

- ESP32
- Servomotores
- Sensor de pulso
- Batería LiPo
