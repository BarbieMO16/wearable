# Wearable
Este sitio documenta los mini proyectos como la pulsera con leds y ademas se documentará por semanas el desarrollo de mi proyecto de wearable.


## Objetivo
Diseñar una prenda interactiva que responda a estímulos mediante sensores y actuadores.


## Componentes utilizados
- ESP32
- Servomotores
- Sensor de pulso
- Batería LiPo
